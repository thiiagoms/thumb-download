
# Youtube Thumb

Dowload your favorite Youtube thumbnails

## Screenshots

![App Screenshot](https://snipboard.io/fXnF8M.jpg)

## Dependencies

- Docker :whale:

## Install :rocket:

1. Clone the project

```bash
$ git clone https://github.com/thiiagoms/thumb-download thumb
```

2. Go to the project directory

```bash
$ cd thumb
```

3. Setup app container and install dependencies:

```bash
thumb $ docker compose up -d 
thumb $ docker compose exec app bash
root@fced9d42c466:/var/www# compose install
```

4. Execute `thumb`
```bash
root@fced9d42c466:/var/www# ./thumb
```
